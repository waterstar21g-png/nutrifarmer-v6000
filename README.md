# nutrifarmer-v6000

**탁월한 찬사 — V6000 모바일 전용**

V5000과 동일 DB·API를 쓰는 모바일 우선 블로그 UI입니다.

## 빠른 시작

```powershell
cd "D:/함께온라인/My_BLOG/nutrifarmer-v6000"
copy ..\nutrifarmer-v5000\.env.local .env.local
npm install
npm run dev
```

브라우저: http://localhost:6000
